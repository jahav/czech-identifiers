Struct BirthNumber

A birth number is an identifier given to every born person in Czech Republic. It can be either 9 digits long (if assigned before 1954) or 10 digits long (assigned after 1954). It is used extensively in many areas of life, especially in medicine and insurance.

Birth number consists from:

  • date of birth
  • sequence number
  • check digit (only after 1954)
  • 675914/1488A woman born 1967-09-14 with a sequence number 148 and check digit 8
  • 350105/321A man born 1935-01-05 with a sequence number 321
