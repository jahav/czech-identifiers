AccountNumber

An account number used in Czech Republic, it consists from three parts:

  • Prefix (optional)
  • Account number
  • Bank code
and generally between prefix and account number is a dash, while between account number and bank code is a slash. 19-123457/0710

BirthNumber

A birth number is an identifier given to every born person in Czech Republic. It can be either 9 digits long (if assigned before 1954) or 10 digits long (assigned after 1954).

Birth number consists from:

  • date of birth
  • sequence number
  • check digit (only after 1954)

IdentificationNumber

Identifier number of a legal person in Czech Republic (also called IČO), either a company or a self-employed trader.
