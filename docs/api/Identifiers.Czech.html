AccountNumber

A bank account number used in Czech Republic, it consists from three parts:

  • Prefix (optional)
  • Account number
  • Bank code
The leading zeros are insignificant, so 00123 is same as 123.

19-123457/0710 is a bank account number with a prefix 19, account number 123457 and a bank code 0710.

0025478/0710 is a bank account number with a prefix 0, account number 25478 and a bank code 0710.

The prefix and number should be visibly separated, but in practice everyone uses hypen. The number is and generally between prefix and account number is a dash, while between account number and bank code is a slash.

BirthNumber

A birth number is an identifier given to every born person in Czech Republic. It can be either 9 digits long (if assigned before 1954) or 10 digits long (assigned after 1954).

Birth number consists from:

  • date of birth
  • sequence number
  • check digit (only after 1954)

IdentificationNumber

Identifier number of a legal person in Czech Republic (also called IČO), either a company or a self-employed trader.

IPattern<TValue>

A string pattern of a type , used to

  • parse a string and create value from the string.
  • create a string from the value.

The implementation of the interface ensure that both parsing and formatting are using same string pattern, although parsing can be more generous with accepted input.
